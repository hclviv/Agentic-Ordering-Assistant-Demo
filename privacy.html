Privacy Policy – Order Assistant (Demo)

Last updated: March 2026

Overview

Order Assistant (Demo) is a demonstration AI assistant designed to showcase conversational food ordering capabilities for Subway. This assistant is intended for demonstration and evaluation purposes only.

Information We Collect

  • ZIP code (for store lookup)
  • Menu selections and customization choices (e.g., items, modifiers)
  • Session-level identifiers (such as a temporary cart ID)

No payment details or personal contact information are collected or stored.

How Information Is Used

  • Simulate a food ordering flow
  • Retrieve mock or demo menu data
  • Create a temporary demo cart
  • Generate a demo checkout link

Data Storage and Retention

No user data is permanently stored. Data is used only for the duration of the demo session.

Payments

This assistant does not process payments and does not collect financial information.

Contact

For questions about this demo or its privacy practices, please contact the demo owner.
